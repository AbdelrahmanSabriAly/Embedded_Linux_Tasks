# Debugging using GDB
### In this part, a GDB script is written and utilized to print the tokens generated in the main program.

<br></br>

# To debug using GDB:
```
gcc -g main.c utils.c -o output
gdb ./output
```

<br></br>

# To use the GDB script, type:
```
source Debugging_Using_GDB/GDB_script
print_tokens
```
